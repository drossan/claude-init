# Skill: SQLite (sqlite)

## Propósito
Utilizar SQLite como base de datos ligera para entornos de desarrollo y testing.

## Responsabilidades
- Configurar SQLite en memoria para la ejecución rápida de tests.
- Asegurar la compatibilidad de los esquemas de TypeORM entre MySQL y SQLite.
- Gestionar la persistencia en archivos locales para desarrollo rápido.
- Resolver discrepancias de tipos de datos entre SQLite y el motor de producción.
- Optimizar la carga de datos semilla (seeds) en bases de datos SQLite.
