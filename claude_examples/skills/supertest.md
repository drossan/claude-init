# Skill: Supertest (supertest)

## Propósito
Realizar pruebas de integración HTTP para validar las rutas y controladores de la API.

## Responsabilidades
- Simular peticiones HTTP (`GET`, `POST`, `PUT`, `DELETE`) al servidor Express.
- Validar códigos de estado HTTP y estructuras de respuesta JSON.
- Probar el flujo completo desde la ruta hasta la base de datos (usando base de datos de test).
- Verificar el correcto funcionamiento de los middlewares (autenticación, validación Zod).
- Asegurar que las cabeceras y parámetros se procesan correctamente.
