# Skill: Experto en TypeScript/Node.js (fullstack-ts-expert)

## Propósito
Aplicar las mejores prácticas del lenguaje TypeScript y del entorno de ejecución Node.js para escribir código eficiente, seguro y mantenible.

## Responsabilidades
- Garantizar el uso de tipado estricto, genéricos y utilidades avanzadas de TypeScript (`Omit`, `Pick`, `Partial`, `Readonly`).
- Optimizar el rendimiento de las aplicaciones Node.js (manejo de asincronía, streams, event loop).
- Asegurar que el código siga las convenciones de estilo establecidas (camelCase para archivos, PascalCase para clases).
- Gestionar y mantener las dependencias del proyecto de forma segura.
- Implementar patrones de programación funcional y declarativa cuando sea apropiado.
- Revisar el uso de `strict: true` en la configuración de TypeScript y evitar el uso de `any`.

## Skills Tecnológicas Clave
- **typescript**: Maestría en el lenguaje y su compilador.
- **node-js**: Conocimiento profundo del runtime y sus APIs core.
