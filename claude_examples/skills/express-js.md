# Skill: Express.js (express-js)

## Propósito
Desarrollar y mantener servidores HTTP robustos utilizando el framework Express.

## Responsabilidades
- Definir rutas claras y organizadas utilizando `express.Router`.
- Implementar middlewares para autenticación, autorización y logging.
- Gestionar correctamente el ciclo de petición y respuesta (`req`, `res`, `next`).
- Configurar el manejo global de errores y respuestas consistentes.
- Utilizar middlewares estándar (body-parser, cors, helmet) de forma segura.
- Asegurar que la lógica de negocio esté desacoplada de los controladores de Express.
