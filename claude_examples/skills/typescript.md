# Skill: TypeScript (typescript)

## Propósito
Garantizar un código robusto, tipado y seguro utilizando TypeScript en su configuración más estricta.

## Responsabilidades
- Mantener `strict: true` en la configuración de TypeScript.
- Definir interfaces y tipos para cada contrato de datos (DTOs, Entidades, Payloads).
- Evitar el uso de `any`, prefiriendo `unknown` o tipos genéricos cuando sea necesario.
- Utilizar utilidades de tipos de TS como `Readonly`, `Partial`, `Pick`, `Omit`.
- Implementar tipos de guarda (type guards) y aserciones de tipo seguras.
- Asegurar que no existan errores de compilación antes de proceder a la ejecución o testeo.
