# Skill: Node.js (node-js)

## Propósito
Optimizar el uso del entorno de ejecución Node.js para asegurar el rendimiento y la escalabilidad de la API.

## Responsabilidades
- Gestionar correctamente la asincronía (async/await, promesas).
- Evitar el bloqueo del Event Loop con operaciones intensivas.
- Utilizar módulos nativos de Node.js cuando sea eficiente (fs, path, etc.).
- Gestionar variables de entorno y configuración del sistema.
- Optimizar el manejo de memoria y procesos (streams, buffers).
- Asegurar la compatibilidad con la versión de Node.js especificada en el proyecto.
