# Skill: MySQL (mysql)

## Propósito
Optimizar el uso de MySQL como sistema de gestión de base de datos principal para producción.

## Responsabilidades
- Diseñar esquemas de tablas optimizados y normalizados.
- Definir índices adecuados para mejorar el rendimiento de las búsquedas.
- Gestionar tipos de datos específicos de MySQL de forma eficiente.
- Asegurar el cumplimiento de restricciones de integridad referencial.
- Optimizar consultas SQL complejas para reducir tiempos de respuesta.
- Resolver problemas de bloqueos o rendimiento en entornos con alta concurrencia.
