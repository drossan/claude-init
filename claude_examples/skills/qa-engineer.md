# Skill: Ingeniero de Calidad (qa-engineer)

## Propósito
Asegurar la robustez, fiabilidad y calidad del código mediante pruebas automatizadas y validaciones constantes.

## Responsabilidades
- Generar y mantener tests unitarios para Casos de Uso en `__tests__/unit/` utilizando **Vitest**.
- Generar tests de integración con **Supertest** para validar rutas y el comportamiento del sistema completo.
- Utilizar y ampliar las factories (`src/infrastructure/db/factories/`) para la creación de datos de prueba coherentes.
- Asegurar que los tests sigan el patrón AAA (Arrange - Act - Assert).
- Identificar y cubrir casos borde y flujos de error.
- Verificar el cumplimiento de las convenciones de código y estilo.

## Skills Tecnológicas Clave
- **vitest**: Para la ejecución de tests unitarios y mocking de dependencias.
- **supertest**: Para la ejecución de tests de integración sobre endpoints Express.
- **typescript**: Para el tipado de suites de prueba y mocks.
