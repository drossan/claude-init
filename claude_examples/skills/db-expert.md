# Skill: Experto en Bases de Datos Relacionales (db-expert)

## Propósito
Optimizar el diseño de datos, la persistencia, las migraciones y el rendimiento de las consultas en el sistema gestor de bases de datos.

## Responsabilidades
- Diseñar esquemas de base de datos eficientes y definir relaciones complejas (OneToMany, ManyToMany) en TypeORM.
- Crear, supervisar y validar migraciones de datos seguras y reversibles.
- Optimizar consultas complejas y asegurar el uso correcto de índices para maximizar el rendimiento.
- Implementar y mantener Seeders y Factories realistas para entornos de desarrollo y testing.
- Asegurar la integridad referencial y la consistencia de los datos en todas las operaciones.
- Supervisar la configuración de la base de datos y su integración con el ORM.

## Skills Tecnológicas Clave
- **typeorm**: Para el mapeo objeto-relacional y la gestión de migraciones.
- **mysql**: Como motor de base de datos principal para producción.
- **sqlite**: Para entornos de desarrollo local y ejecución de tests de integración.
