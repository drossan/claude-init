# claude-init Examples

This directory contains example projects initialized with claude-init.

## Available Examples

### Go Project

A simple Go CLI application with the `.claude/` directory initialized.

**Location**: `examples/go-project/`

**Features**:
- Cobra-based CLI
- Standard Go project layout
- Pre-configured .claude/ directory

**Run**:

```bash
cd examples/go-project
../bin/claude-init detect
```

## Using These Examples

Each example project demonstrates:

1. **Automatic Detection**: How claude-init detects the project type
2. **Generated Structure**: The `.claude/` directory structure
3. **Templates**: Customized agents, skills, and commands

### Quick Start

```bash
# Build claude-init first
cd ..
make build

# Navigate to an example
cd examples/go-project

# See what would be detected
../bin/claude-init detect

# Re-initialize (use --force to overwrite)
../bin/claude-init init --force
```

## Project Structure

Each example follows this structure:

```
example-project/
├── .claude/              # Generated by claude-init
│   ├── agents/           # AI agent definitions
│   ├── skills/           # Language/framework skills
│   ├── commands/         # Custom commands
│   ├── development_guide.md
│   └── .gitignore
├── src/                  # Source code
├── tests/                # Tests
├── package.json/go.mod   # Dependencies
└── README.md
```

## Creating Your Own Example

To create a new example:

1. Create a new directory: `examples/your-project/`
2. Initialize with claude-init: `claude-init init`
3. Add sample code
4. Update this README with your example details

## Tips

- Use `claude-init detect --verbose` to see detailed detection info
- Use `claude-init init --dry-run` to preview what would be generated
- Check the `.claude/development_guide.md` for project-specific guidance
