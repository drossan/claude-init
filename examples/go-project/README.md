# Example Go Project

This is an example Go project initialized with claude-init.

## Project Structure

```
go-project/
├── main.go
├── go.mod
├── go.sum
└── .claude/           # Generated by claude-init
    ├── agents/
    ├── skills/
    ├── commands/
    ├── development_guide.md
    └── .gitignore
```

## Running claude-init

From this directory:

```bash
# Detect project type
../../bin/claude-init detect

# Initialize .claude/ directory
../../bin/claude-init init

# Or with force to overwrite
../../bin/claude-init init --force
```

## Running the Application

```bash
go run main.go
```

Output:
```
Hello, World!
```
