# Example Node.js Project

This is an example Node.js/TypeScript project initialized with claude-init.

## Project Structure

```
nodejs-project/
├── package.json
├── tsconfig.json
├── src/
│   └── index.ts
└── .claude/           # Generated by claude-init
    ├── agents/
    ├── skills/
    ├── commands/
    ├── development_guide.md
    └── .gitignore
```

## Running claude-init

From this directory:

```bash
# Detect project type
../../bin/claude-init detect

# Initialize .claude/ directory
../../bin/claude-init init

# Or with force to overwrite
../../bin/claude-init init --force
```

## Running the Application

```bash
npm install
npm start
```

Output:
```
Hello, World!
```
