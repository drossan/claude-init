# Example Python Project

This is an example Python project initialized with claude-init.

## Project Structure

```
python-project/
├── pyproject.toml
├── requirements.txt
├── src/
│   └── main.py
└── .claude/           # Generated by claude-init
    ├── agents/
    ├── skills/
    ├── commands/
    ├── development_guide.md
    └── .gitignore
```

## Running claude-init

From this directory:

```bash
# Detect project type
../../bin/claude-init detect

# Initialize .claude/ directory
../../bin/claude-init init

# Or with force to overwrite
../../bin/claude-init init --force
```

## Running the Application

```bash
pip install -r requirements.txt
python src/main.py
```

Output:
```
Hello, World!
```
